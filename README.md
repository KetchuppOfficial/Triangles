# HW3D - triangles

This project is a task on course "Uses and applications of C++ language" by [K.I.Vladimirov](https://github.com/tilir). The task was to fast intersect triangles in 3D.

# Building
```bash
cmake -B build
cmake --build build
ctest --test-dir build
```
